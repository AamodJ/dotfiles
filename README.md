# dotfiles

Here you'll find configs for:
1. Window manager: `mangowm`
2. Status bar: `waybar`
3. Notification client: `dunst`
4. Lockscreen: `hyprlock`
5. Idle manager: `hypridle`
6. Theme: `kanagawa-wave`
7. Miscellaneous things to make the setup work (such as custom scripts in `.local/bin`, `.local/sbin` and `.local/scripts`)
8. Portals: `xdg-desktop-portal-hyprland`

Along with the setups, there's common configs for:
1. Terminal: `kitty`
2. Text editor: `nvim`
3. File manager: `lf`
4. Fetch: `pfetch`
5. Shell: `zsh`
